Angel工作室CMS简称Angelcms（企业网站系统）Angelcms!NT_V1.1

主要功能有:

1、企业介绍 及其原asp系统单页面管理

2、新闻信息 asp系统自带功能，新增了可以推荐、热门、置顶、幻灯状态等一些新闻的特性。

3、产品列表 类似asp系统，新增支付、会员价格、推荐、热门、置顶、幻灯状态等操作。

4、图片分享  类似asp系统案例和相册 单是比它强大很多。

5、资源下载  类似下载系统资料下载功能  Angel工作室asp系统没有的特性。

6、留言反馈  客户可以在线留言。

Angelcms!NT运行环境：

服务器操作系统：Windows Server 2000 以上版本

目前支持微软MSSQL/Access数据库：SQL Server 2005 以上版本/Access2000

IIS信息管理器：IIS 6.0 及以上版本(推荐 IIS7.0/8.0)

Angelcms!NT特色功能：

1、asp.net网站管理系统与asp网站管理系统进行比较:这些功能都与企业站一一对应，在后台还可以进行新增其它类似以上功能的栏目，Angel工作室asp.net内容管理系统比asp网站管理系统更加灵活易用。

2、后台全新设计，与前asp网站管理系统不同如：该系统支持伪静态、纯静态以及动态页面后台都可以切换，以及系统采用模板分离原则。

3、Angelcms实现了无限级板块分类，方便内容划分；无限级子栏目分类，清晰展现多级内容。采用符合 DIV+CSS规范布局。

4、网站采用三种页面处理技术，可以生成纯静态网页模式、在不使用asp.net其余为静态插件的前提进行伪静态方式访问、前两者如都不用还可以用动态访问。

5、Angelcms在显示列表数量时，页面可选生成静态，大大减轻服务器的负担、没有造成资源浪费，经过几百万数据量的测试结果显示，加快了浏览的访问速度。

6、Angelcms可根据上传多张图片，生成自定义的预览小图，避免浏览小图情况下加载大图片页面缓慢，减少用户等待的时间，让浏览者更加爽快。

7、Angelcms在用户权限设置方面，页面更好的管理，让站长可以管理多个管理员，每个管理员可以管理自己应该管理的信息，让建站管理起来更轻松。

6、Angelcms模板分离原则，大大减少了站长学习asp.net的困难，在只懂得 HTML 和 CSS 就可以轻松的自定义模版风格。

7、Angelcms采用了jquery+Ajax技术，在无刷新的情况下前台页面的数据进行动态改变，让您快感体验。

8、系统安装简单，只要支持以上运行环境。直接把编译好的程序传入空间中，点击安装文件夹文件运行即可快速搭建好该网站系统。

9、空间只需要支持Access数据就可以运行，对空间要求最低。使得网站建设者不需要购买其他数据库空间的不必要麻烦。
